# 네이버 플레이스 등록 가이드

네이버에서 "대전세븐나이트" 검색 시 **지도와 함께 통합검색 1위에 노출**되려면 네이버 플레이스 등록이 필수입니다.

## 누가 등록할 수 있나

| 입장 | 가능 여부 |
|---|---|
| 사업자(점주) | ✅ 사업자등록증 제출 후 직접 등록 |
| 웨이터(W.T) | ❌ 매장으로 등록 불가, 단 **개인 사업자 등록 시** 가능 |
| 일반인 | ⚠️ "장소 제보"는 가능하지만 운영 권한 없음 |

## 1️⃣ 점주 직접 등록 (사업자등록증 보유 시)

**1단계** — https://smartplace.naver.com 접속, 네이버 ID 로그인
**2단계** — "내 업체 등록하기" → 사업자등록증 업로드
**3단계** — 업체 정보 입력:
- 업체명: 대전세븐나이트
- 업종: 나이트클럽 / 유흥주점
- 주소: 정확한 도로명 + 건물명
- 영업시간: 매일 20:00 ~ 05:00
- 전화: 010-3242-1504
- 홈페이지: https://theassetsquare-svg.github.io/seven2/
**4단계** — 사진 업로드 (최소 5장, 외관·내부·룸·메뉴·로고)
- og-square.png 활용 가능
**5단계** — 검수 → 보통 1~3일 후 등록 완료

## 2️⃣ 웨이터 개인 등록 (개인사업자 등록 후)

웨이터 본인이 **부킹·예약 알선 서비스**로 개인사업자 등록 시 가능:

1. 홈택스 → 사업자등록 신청 → 업종: "기타 정보 서비스업"
2. 네이버 플레이스 등록 → 업종 "기타"
3. 상호: "대전 W.T 원숭이 예약 안내"
4. 주소: 사업장 주소 (집 주소 가능)

## 3️⃣ 등록 후 최적화 (매우 중요)

| 항목 | 영향 |
|---|---|
| **사진 30장 이상** | 네이버 플레이스 노출 가산점 (1순위 신호) |
| **소식·이벤트 주 1회** | 활성 매장 신호, 노출 우선 |
| **방문자 리뷰 30건+** | "리뷰 많은 곳" 검색 결과 상위 |
| **영수증 리뷰 유도** | 가짜 리뷰보다 영수증 인증이 압도적으로 강함 |
| **블로그 리뷰 연동** | 네이버 블로그에서 매장 태그 → 자동 연결 |
| **메뉴/가격 등록** | 정보 풍부 가산점 |
| **예약 기능 ON** | "예약 가능" 필터에 노출 |

## 4️⃣ 노출 키워드 작전

업체 등록 시 **"키워드 직접 입력"** 칸에:
```
대전세븐나이트, 둔산동나이트, 대전나이트클럽, 대전부킹, 4인1조, 세븐나이트
```

소개말에는 자연스럽게:
```
대전 둔산동 No.1 나이트클럽 세븐나이트입니다.
4인1조 부킹 시스템으로 운영되며, 24시간 친절상담합니다.
처음 오시는 분들도 W.T가 책임 안내해드립니다.
☎ 010-3242-1504
```

## 5️⃣ 카카오맵·구글 비즈니스도 동시 등록

- **카카오맵**: place.map.kakao.com → 카카오 ID로 누구나 등록 가능
- **구글 비즈니스**: business.google.com → 우편물 인증 (1~2주)

세 곳 모두 등록되면 검색에서 일관된 신호가 나옵니다.

## 6️⃣ 등록 안 되더라도 차선책

사업자 등록이 어려우면 **본 사이트(https://theassetsquare-svg.github.io/seven2/)** 를 네이버 블로그·카페·카카오톡 오픈채팅 자기소개·인스타 프로필 링크 등에 넣어 **외부 백링크 빌딩**으로 우회. 자세한 건 `BACKLINKS.md` 참조.

## 효과 예상

| 항목 | 등록 전 | 등록 후 |
|---|---|---|
| "대전세븐나이트" 통합검색 | 텍스트 결과만 | **지도+사진+리뷰 박스** 상단 노출 |
| 모바일 검색 | 일반 링크 | **"전화하기" 버튼** 자동 표시 |
| 네이버 지도 검색 | X | **핀 표시** 즉시 노출 |
| AI 검색(Bixby/네이버 AI) | 정보 부족 | 정확한 답변 |

플레이스 등록 한 번으로 검색 노출이 **3~5배** 올라갑니다.
